# ART-028: Assumption Register

Iteration: `baseline-v0.1`
Status: draft

## Purpose

Register all key assumptions underlying the civilization framework. Each assumption must identify where it is used, how it can be tested, what signal would indicate it is false, and its current status.

## Assumption Status Legend

- **Open**: Not yet tested.
- **Testing**: Test in progress (pilot, simulation, or empirical investigation).
- **Supported**: Test results support the assumption.
- **Challenged**: Test results challenge the assumption.
- **Falsified**: Test results falsify the assumption; framework element must be redesigned.
