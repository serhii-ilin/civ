# Baseline Research Questions

This file collects the top-level questions for `baseline-v0.1`. Module-specific questions live in `modules/`, relative to the current agent folder.

## System-Level Questions

- RQ-SYS-001: What institutional architecture best satisfies the value constitution while remaining historically and behaviorally plausible?
- RQ-SYS-002: Which design choices are strongly supported by evidence, which are contested, and which are assumptions?
- RQ-SYS-003: What are the most dangerous failure modes for this society?
- RQ-SYS-004: Which value conflicts are unavoidable rather than design oversights?
- RQ-SYS-005: What minimum viable pilot could test the framework without requiring a full society?
- RQ-SYS-006: Which elements must be validated by simulation before implementation?
- RQ-SYS-007: Which elements cannot be validated by simulation and require historical, legal, ethical, or participatory review?

## Baseline Deliverable Goal

Produce the complete artifact set defined in `canon/artifact-structure.md`.
